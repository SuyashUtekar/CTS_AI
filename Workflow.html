Stage 2 Deep Dive — Zone Identification
```
Template-Based FAST
Uses fixed pixel coordinates from RBI CTS-2010 standard. Works for 80% of clean, aligned scans.
# Fixed CTS-2010 coordinates
CTS_ZONES = {
  "payee": (150, 180, 900, 80),
  "amount": (820, 280, 280, 70),
  "micr": (50, 580, 1100, 60)
}
zone = image[y:y+h, x:x+w]
Anchor-Based SMART
Finds printed anchors (₹ symbol, "Pay", "Date" label, MICR band) and calculates zone positions relative to them.
# Find ₹ symbol anchor
result = cv2.matchTemplate(
  image, rupee_template,
  cv2.TM_CCOEFF_NORMED
)
anchor = cv2.minMaxLoc(result)[-1]
amount_zone = image[y:y+70, x:x+280]
YOLOv8 ML Model ROBUST
Object detection model fine-tuned on Indian cheques. Handles rotation, damage, different bank formats. Fallback for tricky scans.
# ML zone detection
model = YOLO("cheque_zones.pt")
results = model(preprocessed_img)

for box in results[0].boxes:
  label = names[int(box.cls)]
  zones[label] = crop(box.xyxy)
```
